# Data Science Salary Dashboard

## Overview
 The Data Science Salary Dashboard is an interactive Excel-based analytics solution designed to explore global salary trends across different data-related job roles, countries, and employment types.
 
## Excel skills Used
- 📈 Charts
- 💼 Formulas and functions
- 🛠️ Data Validation

 ## Data Jobs Dataset
 The dataset used for this project real-world data science job inforamtion from 2023. The dataset is available via my Excel which provides a foundation for analysing data using Excel . It includes detailed information on:
 - 🔍Job titles
 - 💡Salaries
 - 🌍Locations
 - 📚Skills
